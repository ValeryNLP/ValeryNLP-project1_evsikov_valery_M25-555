# Подземелье Забытых Тайн

Текстовый квест на Python с элементами исследования и головоломками. Вам предстоит найти путь через мрачные залы, избежать смертельных ловушек и разгадать тайну древнего сокровища.

## Особенности
*   **Исследование:** Разветвленная карта комнат, включая секретный сад и темницу.
*   **Загадки:** Интерактивные головоломки, открывающие путь к наградам.
*   **Случайные события:** Каждое прохождение уникально благодаря системе рандомных находок и опасностей.

## Требования

- Python 3.11+
- Poetry (для управления зависимостями)

## Установка

Склонируйте репозиторий и установите зависимости:

```bash
make install
# или вручную:
poetry install
```

## Как играть

Запустите игру командой:

```bash
make project
# или вручную:
poetry run project
```

### Доступные команды

| Команда | Действие |
|---------|----------|
| `look` | Осмотреться вокруг |
| `inventory` | Проверить рюкзак |
| `go <направление>` | Идти (north, south, east, west) |
| `take <предмет>` | Взять предмет |
| `use <предмет>` | Применить предмет из инвентаря |
| `solve` | Попытаться разгадать загадку |
| `help` | Показать этот список |
| `quit` | Покинуть игру |

*Вы также можете использовать быстрые команды перемещения: просто введите `north`, `south` и т.д.*

## Структура проекта

```
labyrinth_game_project/
├── labyrinth_game/         # Пакет с исходным кодом
│   ├── __init__.py
│   ├── main.py             # Точка входа и игровой цикл
│   ├── constants.py        # Карта мира и текстовые данные
│   ├── player_actions.py   # Логика действий игрока
│   └── utils.py            # Вспомогательные функции (рандом, загадки)
├── Makefile                # Команды для сборки и запуска
├── README.md
└── pyproject.toml
```
